Deploy Your First Agent

Get started with the AMOSKYS Intelligence Platform in minutes

# Download agent installer
curl -O https://get.amoskys.com/install.sh

# Run installation
sudo bash install.sh --token YOUR_API_TOKEN

# Select perspectives
amoskys-agent enable process network file

# Start collecting
amoskys-agent start
