What AMOSKYS Monitors

Based on your selected coverage, AMOSKYS will collect and analyze:

  • System process execution and behavior patterns
  • Network connection metadata (IPs, ports, protocols)
  • File integrity checksums for critical paths
  • Authentication events (login, sudo, SSH)

Data Handling

  • All telemetry is stored locally on your device by default
  • Cryptographic integrity via Ed25519 signatures and BLAKE2b hash chains
  • No data is transmitted externally without explicit configuration
  • You retain full ownership and control of all collected data

Security Commitment

  • AMOSKYS agents run with minimal required privileges
  • All agent communications are authenticated and encrypted
  • Circuit breaker patterns prevent resource exhaustion
  • You can disable or remove any agent at any time
