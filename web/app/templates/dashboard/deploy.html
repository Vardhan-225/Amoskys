Questions

What does AMOSKYS monitor?

Process execution, network connections, DNS queries, file changes, login attempts, USB devices, persistence mechanisms (LaunchAgents, cron jobs), and more. All mapped to MITRE ATT&CK.

Will it slow down my Mac?

No. AMOSKYS uses under 2% CPU and 150MB RAM. It runs as a background service with no UI, no popups, no interruptions.

What data is collected?

Security telemetry only: process trees, network flows, file hashes, authentication events. No personal files, no browsing history, no keystrokes, no screenshots.

How do I uninstall?

Run: sudo /Library/Amoskys/deploy/install-from-pkg.sh --uninstall

What are the requirements?

macOS 13 (Ventura) or later, Python 3.11+, and admin access (sudo). The installer checks all requirements automatically.
