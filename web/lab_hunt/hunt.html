Break a WordPress site. Watch every move. Keep the report.

Argos fingerprints a WordPress deployment end-to-end, picks the attack plan its stack is known to be weak against, reasons about chainable exploits, and hands you a branded pentest report you can forward to the site owner.

  • Fingerprints CDN (Cloudflare, Sucuri) · WAF (Wordfence, Sucuri, ModSecurity) · PHP · MySQL · core + plugin versions
  • Picks WordPress-tuned probe order: REST auth → SQLi → file upload → object injection → CSRF → SSRF → LFI → XSS → RCE
  • Adapts encoding cascade to the observed WAF (Wordfence = comment_pad + utf8_overlong + case_mutate)
  • Attempts origin-IP discovery (CT-log SAN bleed, SPF) to bypass Cloudflare
  • Chains findings into full exploit paths (LFI → wp-config.php → DB takeover; upload + LFI → RCE)
